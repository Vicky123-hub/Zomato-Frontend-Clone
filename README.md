# Zomato Landing Page Clone

A responsive front-end clone of the Zomato landing page built using semantic HTML5 and custom CSS3[cite: 1, 2]. This project replicates the landing UI, incorporating key design elements such as promotional banners, locality highlights, features grid, and the premium Zomato Gold tier branding.

## 🚀 Features

- **Hero Header Section:** Immersive background layout featuring the core branding, tagline, and prominent logo[cite: 1, 2].
- **Statistics Section:** Clear presentation showcasing platform metrics (3,00,000+ restaurants, 800+ cities, and 3 billion+ orders)[cite: 1].
- **Popular Localities:** A clean, hover-interactive grid layout highlighting prominent areas across Delhi NCR[cite: 1, 2].
- **App Features Preview:** An interactive display mimicking a smartphone UI wrapper showcasing order features like "Schedule your order," "Veg Mode," and "Gourmet"[cite: 1].
- **Zomato Gold Banner:** A styled, premium dark-themed promotional banner highlighting exclusive membership perks and delivery benefits[cite: 1, 2].
- **Multi-column Footer:** Complete, well-structured multi-column navigation menu referencing enterprise and user guidelines[cite: 1, 2].

💻 Getting Started
Follow these simple steps to run the project locally:

1. Clone the repository:

    git clone [https://github.com/Vicky123-hub/zomato-clone.git](https://github.com/Vicky123-hub/zomato-clone.git)

2. Navigate to the directory:

    cd zomato-clone

3. Launch the application:
    Simply open the index.html file in any modern web browser, or serve it using an extension like Live Server in VS Code.
